# Post-Auction Technical Parameters — Field Definitions

This dataset originates from the FCC’s **Post-Auction Technical Parameters** file, which defines the final (post-repack) RF channel assignments and technical attributes for every full-power and Class A TV station in the United States.

Below is a concise explanation of each column.

## Fields

### **FacID**

Unique numeric FCC Facility ID for each station. Use as the primary node identifier.

### **Site**

Internal FCC site index for tower locations.

### **Call**

Station callsign (e.g., KYES-TV).

### **Ch**

**Post-auction RF channel assignment**.

### **PC**

Pre-auction RF channel.

### **City**

City of license.

### **St**

Two-letter U.S. state or territory code.

### **Lat / Lon**

Transmitter coordinates in FCC’s integer-scaled format.

### **RCAMSL**

Radiation Center Above Mean Sea Level (meters).

### **HAAT**

Height Above Average Terrain (meters).

### **ERP**

Effective Radiated Power (kW).

### **DA**

Directional antenna indicator.

### **AntID**

Antenna ID reference number.

### **Az**

Antenna azimuth (degrees).
